# Remained MVP Features

> Versioned File Manager v0.1 이후 남은 MVP 기능 목록
> 마지막 업데이트: 2025-01-18

---

## 완료된 기능 ✅

| 기능 | 설명 |
|------|------|
| 파일 등록 & 추적 | 드래그앤드롭, 파일 선택 |
| 버전 관리 | 생성, 조회, 복원 |
| 변경 감지 | size + mtime + xxHash64 |
| 태그 시스템 | 추가, 삭제, 정규화 |
| 검색 강화 | 이름, 경로, 커밋 메시지, 태그 검색 |
| 즐겨찾기 | ⭐ 토글, 필터링 |
| 사이드바 필터 | All Files, Favorites, Recent, Modified |
| 테마 | 다크/라이트 모드 |
| 설정 저장 | 윈도우 크기, 정렬, 필터 상태 |
| 이벤트 타임라인 | Restore, Verify 등 작업 히스토리 기록/표시 |
| Display Name 편집 | F2 키, 우클릭 메뉴, Inspector에서 이름 변경 |
| Delete 정책 개선 | Archive/Remove/Trash 옵션, 기본값 저장, Archived 필터 |
| Pin 기능 | 버전 Pin/Unpin, 별도 저장소에 복사, 📌 아이콘 표시 |

## 남은 기능 (우선순위순)

### 1. Job Queue (작업 대기열)
**우선순위: 중간**

비동기 작업을 관리하는 시스템.

- [ ] 작업 유형: Hash verify, Pin copy, Restore, Relink scan
- [ ] 진행률 표시
- [ ] 취소/일시정지/재개
- [ ] 팝업 창 UI
- [ ] 동시 실행 수 설정

---

### 2. Relink (파일 재연결)
**우선순위: 낮음**

파일 경로가 변경되었을 때 다시 연결하는 기능.

- [ ] 수동 Relink (파일 직접 선택)
- [ ] 자동 Relink (폴더 스캔)
  - 파일명/확장자 → size/mtime → hash 순 매칭
  - 후보 제시 + 사용자 확인
- [ ] Relink 후 즉시 검증

---

### 3. UI 개선
**우선순위: 낮음**

- [ ] 파일 타입별 아이콘 표시
- [ ] Inspector 패널 레이아웃 개선
- [ ] 버전 카드 디자인 개선
- [ ] 키보드 단축키 추가

---

### 4. Open With (앱으로 열기)
**우선순위: 낮음**

- [ ] 특정 앱으로 열기 옵션
- [ ] "이 앱으로 항상 열기" 지원
- [ ] 파일별 지정 앱 저장

---

### 5. Metadata 수집
**우선순위: 낮음**

파일 타입별 추가 메타데이터 추출.

- [ ] 이미지: EXIF 정보
- [ ] 영상: 미디어 메타데이터 (해상도, 길이 등)
- [ ] Inspector 패널에 메타데이터 표시

---

### 6. FTS5 전체 텍스트 검색
**우선순위: 낮음**

SQLite FTS5를 이용한 고급 검색.

- [ ] FTS5 가상 테이블 생성
- [ ] 검색 속도 최적화
- [ ] 검색어 하이라이팅
